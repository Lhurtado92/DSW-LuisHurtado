HTML

Etiquetas Descripción
< !DOCTYPE html > la cual permite que los navegadores comprendan que un documento se basa en HTML5
< html >< /html > que indican la etiqueta de apertura y cierre de un documento HTML. Si se desea agregar cualquier tipo de elementos, deben estar dentro de dichas etiquetas
< head >< /head > que indican los metadatos respecto a un documento como los enlaces. Esta clase de información brindada sólo es visible para los navegadores.
< title >< /title> la etiqueta que define el título de una web.
< link > utilizada para afiliar recursos que son externos a dicho documento.
< meta > utilizada para poder colocar metadatos como el autor de la web o hasta su descripción. Siempre es usada para datos que no poseen una etiqueta específica.
< style > funciona como medio para ingresar código CSS en el documento
< body >< /body > Incluye todo el contenido que se desee mostrar dentro de la página web.
< nav >< /nav > Define el contenido que estará ubicado dentro de la sección de navegación de la página.
< main >< /main > Permite elegir el contenido principal que posee un documento, siendo único.

CSS

Estilos Descripción
Colors La propiedad de CSS color selecciona el valor de color de primer plano del contenido de elemento de texto y decoraciones de texto.
Backgrounds La propiedad background es un atajo para definir los valores individuales del fondo en una única regla CSS.
borders La propiedad border permite definir en una única regla todos los bordes de los elementos seleccionados.
margins La propiedad CSS margin establece el margen para los cuatro lados.
text es el módulo de CSS que define cómo realizar la manipulación de elementos de texto como los saltos de línea, la justificación, la alineación, la gestión de espacios en blanco y las transformaciones de texto.
fonts es el modulo CSS que define todo lo relacionado con los recursos tipográficos, sus propiedades y como son cargados.
Height La propiedad CSS height especifica la altura de un elemento.
width La propiedad CSS width especifica la anchura del area de contenido de un elemento.
links La pseudo-clase :link CSS representa un elemento que aún no se ha visitado.
lists Las listas se comportan como cualquier otro texto en su mayor parte, pero hay algunas propiedades CSS específicas de las listas que debes conocer y algunas prácticas recomendadas a tener en cuenta.
tables Las propiedades CSS de las tablas son las que nos permiten controlar los estilos de los títulos de la tabla, el tamaño de las celdas, las filas y las columnas o espaciado entre los bordes.
position La propiedad position de CSS especifica cómo un elemento es posicionado en el documento.
Overflow La propiedad CSS overflow especifica: si recortar contenido, dibujar barras de desplazamiento o mostrar el contenido excedente en un elemento a nivel de bloque.
float La propiedad CSS float ubica un elemento al lado izquierdo o derecho de su contenedor, permitiendo a los elementos de texto y en línea aparecer a su costado.
